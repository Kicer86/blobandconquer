Command Line Options

The following commands can be issued to Blob and Conquer from the command line to invoke various options,

-fullscreenStarts the game in fullscreen
-windowStarts the game in windowed mode
-languageSpecifies the language the game should start in
-noaudioTell the game not to use any audio
-monoTells the game to use Mono sound instead of Stereo
-versionPrints version information
--helpPrints extended help information
-mission <mission to load>Specifies a mission to play as found in the data/missions directory
-load <slot number>Specifies the save slot to load
-nomonstersTells the game not to add any enemies in the mission.
-noitemsTells the game not to add any items in the mission.
-allstaticEnemies and item do nothing. Useful for level testing.
